# Quicklist

Aplicação de gerenciamento de lista de compras desenvolvida como desafio da Rocketseat One na formação full-stack. O objetivo é treinar HTML, CSS e JavaScript criando uma interface simples, clara e responsiva para controlar os itens da semana.

## Descrição

O Quicklist é uma versão prática de um gerenciador de lista de compras semanal. A ideia é permitir ao usuário:

- adicionar novos itens à lista;
- marcar itens como concluídos;
- remover itens quando não forem mais necessários;
- obter feedback visual sobre ações realizadas.

O projeto foca em HTML semântico, CSS para layout e estilização e JavaScript para controle de comportamento.

## Objetivos do desafio

- Título do app: **Compras da semana**
- Botão de voltar no topo
- Campo de entrada para novos itens
- Botão para adicionar itens
- Lista de itens exibida em `ul`
- Estrutura de item com checkbox, texto e botão de remover
- Captura do texto do `input`
- Lógica para adicionar item ao clicar no botão
- Limpeza do campo após adicionar item
- Bloqueio de adição de itens vazios
- Remoção de itens com feedback
- Exibição de alerta na remoção
- Checkbox com estilo visual de concluído
- Permitir desmarcar item e voltar ao estado original
- Layout responsivo para mobile
- Carga inicial de itens
- Testes manuais e verificação de console sem erros
- Estilização adequada para usabilidade seguindo design UI/UX do Figma

## Estrutura do projeto

- `index.html` — marcação principal e estrutura do layout
- `style.css` — reset básico e estilos iniciais
- `script.js` — espaço reservado para a lógica de interação com a lista
- `images/` — recursos visuais da interface

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (DOM)

## Como usar

1. Abra `index.html` no navegador.
2. Insira um produto no campo de entrada.
3. Clique em **Adicionar** para incluir o item na lista.
4. Use o checkbox para marcar como concluído ou desmarcar.
5. Clique em **Remover** para excluir um item.

## Resultado esperado

A aplicação deve atender a um fluxo simples e direto:

- interface limpa e responsiva;
- inputs validados para evitar itens vazios;
- controle de estado de cada item;
- feedback visual para ações como remoção e conclusão;
- compatibilidade com desktop e mobile.
